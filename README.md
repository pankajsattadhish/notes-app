# Notes App

A beginner-friendly **Notes Application** built using **HTML, CSS, and JavaScript** to practice core frontend development concepts such as DOM manipulation, event handling, and browser storage.

---

## Project Overview

This application allows users to create, edit, and delete notes directly in the browser.  
Notes are saved using **LocalStorage**, ensuring data persists across page reloads.  
The app also supports **Markdown formatting** for better note readability.

---

## Key Features

- Create, edit, and delete notes
- Markdown support using `marked.js`
- Automatic data persistence with LocalStorage
- Clean and responsive UI

---

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
- Marked.js
- Font Awesome

---

## How to Run

1. Open `index.html` in a web browser
2. Click **Add Note** to start writing

No installation or backend required.
